# sales-performance-dashboard-powerbi
Interactive Power BI dashboard for sales performance analysis

## Project Overview
This project analyzes sales data to track revenue, profit, region-wise and product-wise performance.

## Key Features
- KPI cards for revenue and profit
- Interactive slicers for region and product
- Trend analysis to identify top and underperforming segments

## Tools Used
- Power BI
- Excel

## Insights
- Identified top-performing products contributing majority of revenue
- Highlighted underperforming regions requiring strategic focus
